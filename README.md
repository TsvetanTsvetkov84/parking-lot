# parking-lot

## Task 1: Basic Structure

Objective: Implement the basic structure of a parking lot with parking spots and vehicle classes.

- Parking lot has a fixed number of parking spots
- Parking lot also has vehicles currently placed on certain spots
- Parking spots and vehicles come in different sizes, make sure the classes can support this
- Parking spots have numbers
- Vehicles have name plates (registration numbers)

## Task 2: Parking Spot Allocation

Context: When you stop at the parking entrance and pull a ticket, a led board shows you the spot number that was allocated for your car based on the size.

Objective: Implement the logic to allocate a parking spot based on the size of the vehicle.

- Method should return the allocated parking spot number
- The parking spot should be marked as occupied
